# Movie Ticket Booking System with Phone OTP (Twilio) - Demo Project

This repository contains a minimal full-stack demo:
- backend/ : Spring Boot application (H2 demo, Oracle profile included)
- frontend/: React application (create-react-app skeleton)

Features:
- Phone-based OTP authentication (Twilio integration included; demo prints OTP)
- Movie listing and booking
- Simple entities: User, OTP, Movie, Booking

Follow backend and frontend READMEs to run locally.
